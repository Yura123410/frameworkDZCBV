# Проект приюта-питомника для собак

Данный проект включает в себя 3 основных раздела:

- Работа с питомцами
- Личный кабинет
- Работа с отзывами

Виртуальное окружение используемое для проекта: .venv

1) После настройки виртуального окружения установите зависимости из файла requirements.txt

```bash
  pip install -r requirements.txt
```

2) Заполните .env файл согласно .env_sample
3) Создайте Базу Данный при помощи команды:

```bash
  python manage.py ccdb
```

4) Создайте миграции при помощи

```bash
  python manage.py makemigrations
```

5) Проведите миграции при помощи

```bash
  python manage.py migrate
```

6) Выполните команду для создания пользователей с разным урровнем доступа

```bash
  python manage.py ccsu
```

7) Выполните команду для заполнения БД используя фикстуры

```bash
  python loaddata dogs_fixture.json
  python loaddata reviews_fixture.json
```

8) Выполните команду для запуска Redis

```bash
  redis-server
```

9) Откройте второе окно терминала

```bash
  python manage.py runserver
```

Модели используемые в проекте:

# Dogs с полями:

- name - Кличка;
- breed - Порода;
- photo - Фотография;
- birth_date - Дата рождения;
- is_active - Активность;
- owner - Хозяин;
- views - Просмотры;

# Reviews с полями:

- title - Заголовок;
- slug - URL;
- content - Содержимое;
- created - Создан;
- sign_of_review - Активность
- author - Автор;
- dog - Собака;

# Users с полями:

- email - Эл. почта;
- role Статус пользователя;
- first_name - Имя;
- last_name - Фамилия;
- avatar - Аватар;
- phone - Номер телефона;
- telegram - Аккаунт телеграм;
- is_active - Статус аккаунта;

# Разрешения:

- Реализованы кастомные разрешения для пользователя с ролью MODERATOR
- Смотреть списки пород, собак и отзывов могут все пользователи
- Смотреть подробную информацию о породе, собаке и отзыве могут все пользователи
